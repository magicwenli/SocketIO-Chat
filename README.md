## SocketIO Chat

### Description

A simple Chatroom server including those features:

- User authentication(Login/Sign up)

- Chatroom creation

- Online user/room list

- p2p chat, file share, etc

### Usage

```shell
pip install -r requirements.txt
python wsgi.py
```
